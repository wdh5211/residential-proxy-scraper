## 住宅代理是什么，为什么爬虫非它不可

简单说，住宅代理用的是真实家庭宽带分配出来的 IP 地址——背后可能就是某个普通用户家里的光猫或者手机基站。对目标网站来说，这种 IP 和普通访客没有任何区别；而数据中心代理的 IP 段是公开可查的，稍微上点规模的反爬系统扫一眼 ASN 就知道你是谁。

这也是为什么住宅代理在这几类场景里几乎是刚需：

- **电商比价/选品**：亚马逊、各国本地站点对机房 IP 风控极严，住宅 IP 通行率明显更高

- **本地化 SEO 监控**：不同城市看到的搜索结果不一样，需要真实地理位置的出口 IP

- **社交平台/账号相关采集**：行为指纹检测严格，数据中心 IP 几乎秒封

- **广告验证**：要看到"普通用户视角"的广告投放结果，而不是被识别成机器人后的特殊页面

但住宅代理也不是没有代价——IP 池要大、要纯净、还得支持稳定轮换或粘性会话，这些自己搭基本搭不起来，所以大部分团队最终都会转向专业服务商。
